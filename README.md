# Extended-Untis-Proxy
Extended-Untis-Proxy is a small program that was developed by students of RFGS, in order to unlock new features such as seaching for free rooms with basic untis access.

## How to Help
> [!IMPORTANT]
> You must be student of RFGS (or your code is that good, that it will be approved without being a student) and all your code has to be approved by the SMV.
>
> I order for the code to be approved, you have to tell a member of SMV that you are working on this project and give your github username or better, dm `tsgscraft` over the RFGS discord your username.

Help is appreciated.
To help, simply clone the repository (`Java 21`) and submit a pull request.
